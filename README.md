# 🎯 Système Autonome de Suivi de Cible — Tourelle Pan-Tilt

> Mécatronique · Vision par ordinateur · Python / OpenMV · UART  
> Projet personnel — 2026

## Description

Tourelle 2 axes (Pan-Tilt) capable de détecter et suivre une cible en temps réel grâce à la vision par ordinateur. Le système intègre une chaîne complète allant de la modélisation mécanique à la validation de la boucle de contrôle.
